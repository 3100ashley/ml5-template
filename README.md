# Naruto Snap Lens 


## Description

A Naurto based Snap Lens using p5.js and ml5.js Using event logic to add in a new drawing. 

## To Run

- Using VS Code use the Live Server Extension. 
- [Github pages link] (https://3100ashley.github.io/naruto-snap-lens--ml5/)
